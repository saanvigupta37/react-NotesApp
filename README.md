# React + Vite
A simple and beginner-friendly notes application built using **React** and **Vite**
This project helps practice core React concepts such as components, props, state management using hooks, and basic form handling.

# React NotesApp

A Simple Notes Application Built with React

Live Demo: https://react-notes-app-lake.vercel.app/

React NotesApp is a lightweight web application that allows users to create and manage notes through a clean and minimal interface.
The project focuses on React fundamentals, component-based design, and state management.

# Features

Add and display notes
Simple and intuitive UI
Component-based architecture
Responsive layout

# Tech Stack

Frontend: React
Build Tool: Vite
Styling: CSS
Deployment: Vercel

# Setup
git clone https://github.com/saanvigupta37/react-NotesApp.git
cd react-NotesApp
npm install
npm run dev
